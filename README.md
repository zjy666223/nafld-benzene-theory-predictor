# Theoretical Benzene-NAFLD Risk Calculator

This repository hosts a static GitHub Pages demo for a theory-oriented benzene-related NAFLD risk calculator.

Important:

- This is a reference model for manuscript presentation and layout only.
- It is not derived from the final real-world deployed cohort model.
- It is not intended for clinical use or individual decision-making.

The deployed page provides:

- Interactive input form
- Instant risk estimate
- Risk category
- Contributor ranking
- A polished front-end suitable for reference screenshots
